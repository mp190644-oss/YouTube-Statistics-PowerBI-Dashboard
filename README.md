# YouTube Statistics Dashboard | Power BI

An interactive Power BI dashboard developed using the Global YouTube Statistics dataset to analyze channel performance, subscriber trends, uploads, views, and rankings. The project demonstrates data cleaning, DAX calculations, and business intelligence visualization techniques.

## Key Features

* KPI Cards for Uploads and Rankings
* Views per Subscriber Analysis using DAX
* Line Chart for High Upload Channels
* Radar Chart for Channel Type vs Rank Analysis
* Packed Bubble Chart for Category-wise Subscribers
* Decomposition Tree for Rank-wise Category Analysis
* Interactive Dashboard Published to Power BI Service

## Tools & Technologies

* Power BI Desktop
* Power BI Service
* Power Query
* DAX
* Card Visuals
* Radar Chart
* Packed Bubble Chart
* Decomposition Tree
* Navigation Buttons

## Key Insights

* Identified top-performing YouTube channels based on uploads and views.
* Compared subscriber distribution across categories.
* Analyzed ranking patterns across channel types.
* Evaluated audience engagement using Views per Subscriber.
